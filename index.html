Signature

Audit Log SDK

One init call patches window.fetch across the whole app — every API call is captured, batched, and shipped with actor, action, and status. Works through micro-frontends too. A matching Go middleware records inbound requests, so nothing slips past.

// app/layout.tsx
import { initAudit } from "@bastion/audit-sdk";

initAudit({
  endpoint: "/api/audit",
  flushInterval: 5000,
  batchSize: 20,
});
// that's it — everything is tracked
Pro

Permissions as resource:action, scoped per team. One definition guards your React UI and your Go endpoints — no drift between what a user can see and what they can call.

Before you ask

Questions

Is this a boilerplate or a framework?

A boilerplate. It's your code the moment you clone it — plain Next.js and Go, no hidden runtime, no lock-in. Delete what you don't need.

Why Go instead of a Node backend?

Because most starters give you a toy API that buckles under real traffic. Go with tuned pooling and clean architecture is what you'd actually run in production — and it's rare in this price range.

What auth does it use?

Email + password with an HS256 JWT in an httpOnly cookie, verified at the edge. OIDC / Keycloak integration is on the roadmap.

Can I build client projects with it?

Yes. Pro and Team allow unlimited commercial projects. You just can't resell Bastion itself as a template.

What if I get stuck?

Every tier ships with docs and a structured README. Team adds a setup call and priority email for three months.
